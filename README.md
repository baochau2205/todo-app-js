# Todo App (JavaScript Project)

A simple and interactive Todo App built with **HTML, CSS, and Vanilla JavaScript**.  
This project was created while learning core JavaScript concepts and DOM manipulation.

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API
- DOM Manipulation

---

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Save tasks in local storage
- Persistent data after refresh

---

## What I Learned

- How to manipulate the DOM using JavaScript
- How event handling works (click events)
- How to use arrays and objects to manage data
- How localStorage works for saving data in the browser
- How to structure a small JavaScript project
